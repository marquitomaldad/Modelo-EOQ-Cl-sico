Programa  hecho en Java para resolver modelos básicos de Investigación de Operaciones:

- EOQ Clásico
- EOQ con Descuentos (tres precios)

El sistema calcula:
- Cantidad óptima de pedido (EOQ)
- Duración del ciclo
- N
- Le
- Punto de reorden
- Costos totales

Incluye un menú en consola para elegir entre EOQ clásico o con descuentos.  
El usuario ingresa los datos y el programa muestra los resultados.
